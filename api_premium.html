Endpoint

Criar Usuário

Registra um usuário da Atlântica Natural dentro da ATL Academy. Após a criação, o usuário poderá fazer login no painel ATL com o mesmo e-mail e senha.

POST /auth/v1/admin/users
CampoTipoStatusDescrição
email string obrigatório E-mail único do usuário
password string obrigatório Senha — mínimo 6 caracteres
email_confirm boolean opcional Enviar true para confirmar o e-mail automaticamente (recomendado)
user_metadata.full_name string opcional Nome completo do usuário
Node.js 
const res = await fetch(
  `${SUPABASE_URL}/auth/v1/admin/users`,
  {
    method: "POST",
    headers: {
      "apikey":        ATL_SERVICE_ROLE_KEY,
      "Authorization": `Bearer ${ATL_SERVICE_ROLE_KEY}`,
      "Content-Type":  "application/json"
    },
    body: JSON.stringify({
      email:          "usuario@atlanticanatural.com.br",
      password:       "senha123",
      email_confirm:  true,
      user_metadata:  { full_name: "Nome Completo" }
    })
  }
);
const user = await res.json();
console.log(user.id); // UUID do novo usuário
200 — Criado com sucesso

Endpoint

Listar Usuários

Retorna todos os usuários cadastrados na ATL Academy. Use antes da sincronização em lote para verificar quais usuários já existem.

GET /auth/v1/admin/users
Node.js 
const res = await fetch(
  `${SUPABASE_URL}/auth/v1/admin/users`,
  {
    headers: {
      "apikey":        ATL_SERVICE_ROLE_KEY,
      "Authorization": `Bearer ${ATL_SERVICE_ROLE_KEY}`
    }
  }
);
const { users } = await res.json();
console.log(`Total: ${users.length} usuários`);
200 — OK
JSON Response 
{
  "users": [
    {
      "id":               "uuid",
      "email":            "usuario@email.com",
      "created_at":       "2026-01-15T10:00:00Z",
      "last_sign_in_at":  "2026-04-13T12:30:00Z"
    }
  ],
  "total": 142
}
Script

Sincronização em Lote

Use este script para importar todos os usuários da Atlântica Natural de uma só vez. Substitua o array usuarios com os dados reais exportados da Atlântica Natural.

⚠️
Execução única: Rodar apenas uma vez para a importação inicial. Para novos usuários após isso, usar o endpoint POST /auth/v1/admin/users individualmente a cada novo cadastro.
sync.js — Node.js 
const SUPABASE_URL = "https://xicjkwzwmmlxgnbfgkga.supabase.co";
const KEY = process.env.ATL_SERVICE_ROLE_KEY;

// ← Substituir com os dados reais da Atlântica Natural
const usuarios = [
  { email: "user1@email.com", senha: "senha1", nome: "Nome 1" },
  { email: "user2@email.com", senha: "senha2", nome: "Nome 2" },
];

async function criarUsuario(u) {
  const res = await fetch(`${SUPABASE_URL}/auth/v1/admin/users`, {
    method: "POST",
    headers: {
      "apikey":        KEY,
      "Authorization": `Bearer ${KEY}`,
      "Content-Type":  "application/json"
    },
    body: JSON.stringify({
      email:         u.email,
      password:      u.senha,
      email_confirm: true,
      user_metadata: { full_name: u.nome }
    })
  });
  return res.json();
}

async function sincronizar() {
  console.log(`Iniciando sync de ${usuarios.length} usuários...`);
  for (const u of usuarios) {
    const d = await criarUsuario(u);
    if (d.id)                console.log(`✓  ${u.email}`);
    else if (d.code === 422) console.log(`~  ${u.email} (já existe)`);
    else                   console.error(`✗  ${u.email}`, d.msg);
    await new Promise(r => setTimeout(r, 200));
  }
  console.log("Sync concluído!");
}

sincronizar();
