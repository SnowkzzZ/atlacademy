Endpoint 1 — Criar Usuário

Cria um usuário da Atlântica Natural dentro da ATL Academy. Usar para cada novo usuário que deve ter acesso ao painel.

CampoTipoObrigatórioDescrição
email string Sim E-mail do usuário (deve ser único)
password string Sim Senha (mínimo 6 caracteres)
email_confirm boolean Não Enviar true para confirmar o e-mail automaticamente (recomendado)
user_metadata.full_name string Não Nome completo do usuário
Requisição — POST
URL: POST /auth/v1/admin/users

Body (JSON):
{
  "email": "usuario@atlanticanatural.com.br",
  "password": "senha123",
  "email_confirm": true,
  "user_metadata": {
    "full_name": "Nome do Usuário"
  }
}
Resposta — Sucesso (200 OK)
{
  "id": "uuid-gerado-automaticamente",
  "email": "usuario@atlanticanatural.com.br",
  "created_at": "2026-04-13T17:00:00Z",
  "email_confirmed_at": "2026-04-13T17:00:00Z"
}
Resposta — E-mail já existente (422)
{ "code": 422, "msg": "User already registered" }

Script — Sincronização em Lote (Node.js)

Usar para importar todos os usuários da Atlântica Natural de uma vez só. Substituir o array usuarios com os dados reais.

Node.js
const URL = "https://xicjkwzwmmlxgnbfgkga.supabase.co";
const KEY = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJpc3MiO...";
// Usar a chave completa fornecida na seção Autenticação

const usuarios = [
  { email: "user1@email.com", senha: "senha1", nome: "Nome 1" },
  { email: "user2@email.com", senha: "senha2", nome: "Nome 2" },
];

async function sincronizar() {
  for (const u of usuarios) {
    const res = await fetch(`${URL}/auth/v1/admin/users`, {
      method: "POST",
      headers: {
        "apikey": KEY,
        "Authorization": `Bearer ${KEY}`,
        "Content-Type": "application/json"
      },
      body: JSON.stringify({
        email: u.email,
        password: u.senha,
        email_confirm: true,
        user_metadata: { full_name: u.nome }
      })
    });
    const d = await res.json();
    console.log(d.id ? `OK: ${u.email}` : `ERRO: ${u.email} - ${d.msg}`);
    await new Promise(r => setTimeout(r, 200)); // evitar rate limit
  }
  console.log("Sincronização concluída!");
}

sincronizar();
