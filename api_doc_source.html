Listar Todos os Usuarios da ATL Academy

Retorna todos os usuarios cadastrados no painel da ATL Academy. Util para verificar quais usuarios ja existem antes de sincronizar.

GET
/auth/v1/admin/users
Exemplo — cURL
cURL
curl -X GET \
  "https://xicjkwzwmmlxgnbfgkga.supabase.co/auth/v1/admin/users" \
  -H "apikey: SERVICE_ROLE_KEY" \
  -H "Authorization: Bearer SERVICE_ROLE_KEY"
Exemplo — JavaScript
JavaScript / Node.js
const res = await fetch(
  "https://xicjkwzwmmlxgnbfgkga.supabase.co/auth/v1/admin/users",
  {
    headers: {
      "apikey": "SERVICE_ROLE_KEY",
      "Authorization": "Bearer SERVICE_ROLE_KEY"
    }
  }
);
const { users } = await res.json();
console.log(users);
Resposta de Sucesso — 200 OK
JSON
{
  "users": [
    {
      "id": "uuid-do-usuario",
      "email": "usuario@email.com",
      "created_at": "2026-01-15T10:00:00Z",

Criar Usuario (Sincronizar da Atlantica Natural)

Cria um novo usuario da Atlantica Natural dentro da ATL Academy. Chamar este endpoint para cada usuario que deve ter acesso ao painel ATL Academy.

POST
/auth/v1/admin/users
Parametros do Body (JSON)
CampoTipoStatusDescricao
emailstringobrigatorioEmail do usuario (deve ser unico na plataforma)
passwordstringobrigatorioSenha do usuario (minimo 6 caracteres)
email_confirmbooleanopcionaltrue = email confirmado automaticamente. Recomendado: true
user_metadata.full_namestringopcionalNome completo do usuario
Exemplo — cURL
cURL
curl -X POST \
  "https://xicjkwzwmmlxgnbfgkga.supabase.co/auth/v1/admin/users" \
  -H "apikey: SERVICE_ROLE_KEY" \
  -H "Authorization: Bearer SERVICE_ROLE_KEY" \
  -H "Content-Type: application/json" \
  --data-raw '{
    "email": "joao@atlanticanatural.com.br",
    "password": "senha_do_usuario",
    "email_confirm": true,
    "user_metadata": { "full_name": "Joao Silva" }
  }'
Exemplo — JavaScript
JavaScript / Node.js
const res = await fetch(
  "https://xicjkwzwmmlxgnbfgkga.supabase.co/auth/v1/admin/users",
  {
    method: "POST",
    headers: {
      "apikey": "SERVICE_ROLE_KEY",
      "Authorization": "Bearer SERVICE_ROLE_KEY",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      email: "joao@atlanticanatural.com.br",
      password: "senha_do_usuario",
      email_confirm: true,
      user_metadata: { full_name: "Joao Silva" }
    })
  }
);
const data = await res.json();
console.log("Usuario criado:", data.id);
Resposta de Sucesso — 200 OK
JSON
{
  "id": "novo-uuid-gerado",
  "email": "joao@atlanticanatural.com.br",
  "created_at": "2026-04-13T17:00:00Z",

Sincronizar Lista Completa em Lote

Script completo para importar todos os usuarios da Atlantica Natural de uma so vez. Executar no servidor da Atlantica Natural.

Como usar

Substituir o array "usuarios" pelos dados reais exportados do sistema da Atlantica Natural e executar o script uma unica vez para a sincronizacao inicial. Apos isso, usar o endpoint POST individualmente para novos cadastros.

Node.js — Sync em Lote
const SUPABASE_URL    = "https://xicjkwzwmmlxgnbfgkga.supabase.co";
const SERVICE_ROLE_KEY = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...";

// Substituir com os usuarios reais da Atlantica Natural
const usuarios = [
  { email: "usuario1@email.com", senha: "senha1", nome: "Usuario 1" },
  { email: "usuario2@email.com", senha: "senha2", nome: "Usuario 2" },
  // adicionar quantos usuarios precisar
];

async function criarUsuario(user) {
  const res = await fetch(`${SUPABASE_URL}/auth/v1/admin/users`, {
    method: "POST",
    headers: {
      "apikey": SERVICE_ROLE_KEY,
      "Authorization": `Bearer ${SERVICE_ROLE_KEY}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      email: user.email,
      password: user.senha,
      email_confirm: true,
      user_metadata: { full_name: user.nome }
    })
  });
  return res.json();
}

async function sincronizarTodos() {
  console.log(`Iniciando sync de ${usuarios.length} usuarios...`);

  for (const user of usuarios) {
    const data = await criarUsuario(user);

    if (data.id) {
      console.log(`[OK]  ${user.email}`);
    } else if (data.msg === "User already registered") {
      console.log(`[JA EXISTE]  ${user.email}`);
    } else {
      console.error(`[ERRO]  ${user.email}`, data);
    }

    // Aguardar 200ms para evitar rate limit
    await new Promise(r => setTimeout(r, 200));
  }

  console.log("Sincronizacao concluida!");
}

sincronizarTodos();
